# 第一轮裁剪目标

## 保留前提
这一轮明确不动：
- 软件商店
- 多语言包
- OpenClash
- 旁路由相关网络栈
- LuCI 核心

## 删除范围

### 1. 多媒体
目标：删除音视频播放、媒体服务、转码类无关组件。

重点关注：
- minidlna / dlna 服务
- ffmpeg 相关组件
- 音视频工具链
- 媒体库服务

### 2. 打印
目标：删除打印服务和驱动层用户态组件。

重点关注：
- cups
- p910nd
- 打印机管理前端

### 3. 蓝牙
目标：删除所有蓝牙栈和工具。

重点关注：
- bluez
- bluetoothd
- 蓝牙 CLI 工具

### 4. 下载器
目标：删除下载类应用与对应 LuCI 页面。

重点关注：
- aria2
- qBittorrent
- transmission
- 迅雷/其他下载器（如镜像中存在）

### 5. 多余主题
目标：仅保留 1 个默认主题，其它主题先删。

重点关注：
- luci-theme-* 多个并存时，只留当前默认主题
- 不删除 luci-base 共享资源

## 执行建议
1. 先在容器里导出已安装包列表。
2. 从包列表里筛这 5 类目标。
3. 生成候选删除清单。
4. 删除前先备份包列表。
5. 每删一类后做页面与网络冒烟测试。
