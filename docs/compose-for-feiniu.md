# 飞牛场景下的 Compose 建议

## 推荐部署方式
如果你要以 Compose 方式部署，这一版建议是：
- 自定义构建镜像：`istoreos-fn:minimal-v1`
- `privileged: true`
- `macvlan` 独立 IP
- 固定 `ipv4_address`
- 预留配置持久化目录

## 为什么用 Compose
1. 结构清晰，适合飞牛导入/复用。
2. 网络和卷定义集中管理。
3. 后续升级镜像、替换 tag、增加挂载更方便。

## 后续可继续优化
- 把 `image:` 改成仓库镜像地址
- 调整持久化挂载到更精确目录
- 增加健康检查
- 增加只读根文件系统（若后续验证可行）
