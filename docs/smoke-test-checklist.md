# 冒烟测试清单

## 1. 容器基础
- 容器能正常启动
- `ps` 可见 `procd` / `rpcd` / `uhttpd`
- Web 管理地址可访问

## 2. LuCI
- 首页正常打开
- 系统菜单正常
- 网络菜单正常
- 不出现 500/空白页

## 3. 软件商店
- `luci-app-store` 页面可打开
- 应用列表可加载
- 页面不报依赖错误

## 4. OpenClash
- `luci-app-openclash` 页面可打开
- 插件配置可保存
- 核心可识别或可下载
- 服务可启动

## 5. 旁路由能力
- 容器 macvlan IP 可访问
- DNS 正常解析
- 客户端将网关/DNS 指向容器后能上网
- OpenClash 开启后策略路由仍可工作

## 6. 主题与多语言
- 默认主题显示正常
- 至少一种中文界面正常
- 删除多余主题后无静态资源缺失

## 7. 失败回滚
- 保留删除前包列表
- 保留删除脚本执行日志
- 删除失败时可恢复到原镜像重试
